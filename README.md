# ETH数据分析仪表板

这是一个用于分析以太坊(ETH)交易量和价格预测的可视化仪表板。

## 功能特性

1. **📊 历史交易量分析**: 展示2024年9月1日到9月22日ETH的详细交易量数据
2. **🔮 价格预测**: 基于历史数据演算出9月23日到9月28日最合理的K线预测
3. **🏛️ 交易所分析**: 显示各大交易所的交易量市场份额
4. **📈 可视化图表**: 交互式K线图和饼图展示

## 数据来源

- 基于真实的ETH价格趋势和市场数据生成模拟数据
- 包含22天的历史数据和6天的预测数据
- 涵盖8大主要交易所的交易量信息

## 快速开始

### 方法1: 使用启动脚本
```bash
./start_server.sh
```

### 方法2: 手动启动
```bash
# 1. 生成数据
python3 eth_data_simple.py

# 2. 启动服务器
python3 app_simple.py
```

## 访问仪表板

在浏览器中打开: http://localhost:8000

## 技术架构

- **后端**: Python3 内置HTTP服务器
- **前端**: HTML5 + CSS3 + JavaScript
- **图表库**: Chart.js
- **数据格式**: JSON

## 文件结构

```
/workspace/
```

## API端点

- `GET /` - 主页面
- `GET /api/data` - 所有数据
- `GET /api/historical` - 历史K线数据
- `GET /api/prediction` - 预测K线数据
- `GET /api/exchanges` - 交易所数据
