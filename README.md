# ETH交易量和凯利指数分析平台

一个基于Web的以太坊(ETH)交易量分析和凯利指数计算平台，提供各大交易所的实时数据可视化和智能投资建议。

## 🌟 功能特性

### 📊 数据获取
- **各大交易所ETH交易量**: 获取Binance、Coinbase、Kraken、OKX等主流交易所的ETH交易量数据
- **历史数据分析**: 过去15天的详细交易数据
- **实时数据更新**: 支持手动刷新和自动更新

### 🎯 凯利指数分析
- **智能计算**: 基于凯利公式计算最优投资比例
- **风险评估**: 分析各交易所的投资风险和收益潜力
- **投资建议**: 提供基于数据的投资建议

### 📈 可视化界面
- **交互式图表**: 使用Chart.js创建美观的数据可视化
- **响应式设计**: 支持桌面和移动设备
- **实时更新**: 数据自动刷新，保持最新状态

## 📊 凯利指数说明

### 凯利公式
```
f* = (bp - q) / b
```

其中：
- `f*` = 建议的资金投入比例
- `b` = 赔率 (平均盈利/平均亏损)
- `p` = 获胜概率 (上涨概率)
- `q` = 失败概率 (下跌概率) = 1 - p

### 投资建议等级
- **强烈建议投资** (凯利指数 > 0.15)
- **建议投资** (凯利指数 > 0.10)
- **谨慎投资** (凯利指数 > 0.05)
- **少量投资** (凯利指数 > 0)
- **不建议投资** (凯利指数 ≤ 0)

## ⚠️ 风险提示

1. **数据来源**: 当前使用模拟数据，实际使用时请接入真实API
2. **投资风险**: 凯利指数基于历史数据，不保证未来表现
3. **市场波动**: 加密货币市场波动性极大，存在重大损失风险
4. **投资建议**: 仅供参考，请结合基本面分析综合判断
5. **风险控制**: 建议使用保守的凯利分数(如1/2或1/3凯利)
